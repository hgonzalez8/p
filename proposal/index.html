Summary

This project we plan to extend a basic path tracer to model volumetric light interactions—such as fog, mist, and haze—by incorporating scattering and absorption within the space between surfaces, enabling more realistic effects like softer shadows and visible light beams. The team will compare standard path tracing with more advanced methods (like bidirectional approaches) in terms of runtime and visual quality across different scenes. If time permits, we aim to build an interactive GUI that allows real-time exploration of the rendered environments.
